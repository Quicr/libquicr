Detailed Description

MOQ track handler for published track.

MOQ publish track handler defines all track related callbacks and functions for publish. Track handler operates on a single track (namespace + name).

This extends the base track handler to add publish (aka send) handling

Publish object to announced track.

Publish object to announced track that was previously announced. This will have an error if the track wasn't announced yet. Status will indicate if there are no subscribers. In this case, the object will not be sent.

Parameters
[in]group_idGroup ID of object
[in]object_idObject ID of the object
[in]objectObject to publish to track
[in]ttlExpire TTL for object
[in]priorityPriority for object; will be set upon next qualifing stream object
