Detailed Description

MOQ track delegate for subscribe and publish.

MOQ track delegate defines all track related callbacks and functions. Track delegate operates on a single track (namespace + name). It can be used for subscribe, publish, or both subscribe and publish. The only requirement is that the namespace and track alias be the same.

std::optional<uint64_t> quicr::MoQTrackDelegate::_subscribe_id
protected

_subscribe_id is the primary index/key for subscribe subscribe context/delegate storage. It is use as the subscribe_id in MOQT related subscribes. Subscribe ID will adapt to received subscribe IDs, so the value will reflect either the received subscribe ID or the next one that increments from last received ID.
